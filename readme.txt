== Description ==

This plugin allows you to work on your website and have a custom splash page at the same time.

Only administrators can see your website. Or if you wish to like it to show to a client, you can send them the link with: www.example.com/?splash=off


== Installation ==

1. Upload `wp-splash-light` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy "splash-page.template.php" to your custom theme, and you are ready to go.
4. To disable the splash page, simply deactivate the plugin.

== Frequently asked questions ==

= How do I know that it works? =

To see if it works, please logout. And you will see the splash page.

= How can I show the template to the client? =

Send your client the link www.example.com/?splash=off and the client should be able to see the theme.
